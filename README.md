# k-NN Stroke Prediction Tutorial

This repository accompanies my Machine Learning Tutorial assignment.  
It demonstrates how the **k-Nearest Neighbours (k-NN)** algorithm behaves under **severe class imbalance** using the public **Stroke Prediction Dataset**.  
The project includes a full step-by-step tutorial (PDF) and a reproducible Jupyter notebook showing preprocessing, model training, evaluation, and visualisation.
